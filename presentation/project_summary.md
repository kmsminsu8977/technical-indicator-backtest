# 발표 초안

## 1. 문제 정의

이동평균 교차 신호는 거래비용을 고려해도 단순 보유보다 나은 경로를 만드는가?

## 2. 데이터와 가정

- 합성 샘플 데이터: `data/sample/price_series.csv`
- 재현 가능한 baseline 실행을 우선 구성

## 3. 방법

단기/장기 이동평균 신호를 재현 가능한 백테스트 테이블로 변환한다.

## 4. 현재 산출물

- 실행 스크립트: `python -m src.run_baseline`
- 결과 표: `outputs/tables/baseline_results.csv`

## 5. 후속 작업

- 실제 데이터 연결
- 민감도 분석
- 차트/보고서 산출
- 프로젝트별 상세 검증
